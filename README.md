# Kasir-Restoran-Python
Program ini dibuat untuk mengotomatisasi perhitungan total belanja dengan fitur:
✅ Cek ambang batas diskon otomatis.
✅ Perhitungan persentase pajak secara dinamis.
✅ Logika kontrol alur (If-Else) yang bersih.

### Cara Penggunaan:
1. Jalankan file automated_billing_system.py.
2. Masukkan input harga (jika sudah menggunakan fungsi input).
3. Hasil akhir akan muncul di terminal.
