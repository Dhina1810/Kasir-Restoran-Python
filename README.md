Proyek ini mendemonstrasikan logika pemrograman Python untuk menangani alur transaksi retail/restoran.

### Aturan Bisnis yang Diimplementasikan:
* *Diskon Otomatis:* Program akan memeriksa apakah total belanja melebihi *Rp500.000. Jika ya, diskon sebesar **10%* akan diterapkan.
* *Perhitungan Pajak:* Pajak sebesar *5%* dihitung secara akurat dari harga akhir setelah penyesuaian diskon.
* *Flow Control:* Menggunakan struktur if-else untuk memastikan harga tetap normal jika syarat diskon tidak terpenuhi.
